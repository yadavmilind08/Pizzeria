Follow these steps to set up and run the API project locally:
  1) git clone https://github.com/yadavmilind08/Pizzeria.git
  2) cd API
  3) dotnet build
  4) dotnet run

Follow these steps to set up and run the client project locally:
  1) git clone https://github.com/yadavmilind08/Pizzeria.git
  2) cd client
  3) npm install
  4) ng serve
